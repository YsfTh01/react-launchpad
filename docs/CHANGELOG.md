# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial boilerplate setup with React 19, Vite 7, TypeScript 5.9
- Feature-based architecture with strict import boundaries
- TanStack Query for server state management
- Zustand for client state management
- React Hook Form + Zod for form handling
- i18next for internationalization (EN/TR)
- Tailwind CSS v4 for styling
- Complete UI component library (Button, Input, Modal, DataTable, etc.)
- Dark mode support (Light/Dark/System) with global synchronization
- Vitest + Testing Library for unit testing
- Playwright for E2E testing
- MSW for API mocking
- Storybook for component documentation
- ESLint + Prettier + Husky for code quality
- Commitlint for conventional commits
- GitHub Actions CI/CD workflows
- Sentry integration for error tracking
- PostHog analytics integration
- Feature flags system with DevTools
- Structured logging system
- Web Vitals performance monitoring
- PWA support with offline caching
- Security headers (CSP, X-Frame-Options, etc.)
- API client with retry logic and interceptors
- Bundle analysis with visualizer
- Plop generators for features, components, hooks
- OpenAPI type generation script
- Interactive cleanup script for removing optional features
- Docker support (Dockerfile, docker-compose.yml, nginx.conf)
- Comprehensive documentation (Architecture, Contributing, Security)
